Datenschutzerklärung — FORTRESS

Stand: Juli 2026  |  App-Version: 3.39.1

1. Verantwortlicher

FORTRESS wird als Open-Source-Hobbyprojekt betrieben.
Kontakt: GitHub Issues

2. Welche Daten werden gespeichert?

2.1 Lokal auf deinem Gerät (localStorage)

Das Spiel speichert folgende Daten ausschließlich auf deinem Gerät:

  • Spielername und gewählter Avatar (Wappen)
  • Spielstatistiken (Siege, Niederlagen, ELO-Wertung)
  • Level, Erfahrungspunkte (XP) und Gold
  • Freigeschaltete Achievements
  • Tägliche Belohnungs-Streak

Diese Daten verlassen dein Gerät nicht, solange du kein Online-Spiel spielst.

2.2 Online-Spielbetrieb (Firebase Realtime Database)

Wenn du ein Online-Spiel startest oder der Bestenliste beitrittst, werden folgende Daten temporär in der Google Firebase Realtime Database gespeichert:

  • Spielstatus: Spielfeld, Phase, Runde, Punkte (nur während eines aktiven Spiels, danach automatisch gelöscht)
  • Matchmaking-Warteschlange: Spielername, Avatar-Kennung, ELO-Wertung, Zeitstempel (nur während der Suche, max. ~60 Sekunden)
  • Bestenliste: Spielername, Avatar, ELO-Wertung, Siege, Level (dauerhaft, bis du dein Profil zurücksetzt)

Es werden keine echten Namen, E-Mail-Adressen oder Kontodaten gespeichert. Spielernamen wählst du selbst (Pseudonym empfohlen).

3. Firebase / Google

Die Firebase Realtime Database wird von Google LLC betrieben. Durch das Spielen im Online-Modus gelten zusätzlich Googles Datenschutzbestimmungen für Firebase. Google kann Metadaten (IP-Adresse, Zeitstempel) im Rahmen des Datenbankbetriebs verarbeiten.

4. Keine Werbung, kein Tracking

FORTRESS enthält keine Werbung, kein Analytics und kein Tracking. Es werden keine Daten an Dritte (außer Firebase für Online-Spielbetrieb) weitergegeben.

5. Kinder

Das Spiel richtet sich nicht spezifisch an Kinder unter 13 Jahren. Es werden keine personenbezogenen Daten von Minderjährigen wissentlich erhoben.

6. Deine Rechte

Du kannst deine Daten jederzeit löschen:

  • Lokale Daten: Browsereinstellungen → Websitedaten löschen → fortress (oder App-Daten löschen)
  • Bestenliste: Profil im Spiel zurücksetzen (löscht deinen Eintrag aus der Firebase-Bestenliste)

7. Änderungen dieser Erklärung

Wesentliche Änderungen werden im Changelog auf GitHub angekündigt.
