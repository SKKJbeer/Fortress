Why this works with an AI

House rules, written for human and machine

  • Single source of truth — no feature without a spec entry
  • Test gate — green or it doesn't ship
  • Version discipline — bump, tag, release. Every time.
  • Bug register — solved bugs + root cause + "never again"

Just good engineering culture — written so a machine can follow it. Works on humans, too.
